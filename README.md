# ZiffyHealth — Design Demo

Public preview of the ZiffyHealth **green design system**: the clinical report archetypes and the doctor/patient apps. **Synthetic data only — no real patient information.**

**Demo hub:** https://nilenduganguli.github.io/view.io/

Old URLs preserved: `static.html` → comprehensive report, `prescription.html` → prescription (redirects).

## Notes
- Reference ranges are sourced from the **KIN Diagnostics** reference sheet.
- All surfaces are token-driven (green-only; amber/orange/red reserved for clinical zones) and pass a **WCAG-AA** contrast gate.
- The apps open all three report types; the interactive report uses its sibling `report-final.css` / `report-fix.css`.
